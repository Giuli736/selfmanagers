# 📸 GUIDA ALLE IMMAGINI — SelfManager

Questa guida ti spiega quali immagini caricare, dove metterle e in che formato.

---

## 📁 Struttura cartelle

```
images/
├── hero/              → Foto per le "cartoline" nella Homepage
├── portfolio/
│   ├── social/        → Lavori social media di Giuliano
│   └── web/           → Siti web realizzati da Nicholas
├── clients/           → Loghi dei clienti (marquee)
└── team/              → Foto del team (opzionale)
```

---

## 🏠 HERO / HOMEPAGE — Cartoline con rotazione
📂 Cartella: `images/hero/`

Carica da **3 a 6 foto** con questi nomi esatti:

| Nome file         | Contenuto consigliato                                 |
|-------------------|-------------------------------------------------------|
| `hero-1.jpg`      | Foto di lavoro, scrivania, setup, brainstorming       |
| `hero-2.jpg`      | Screenshot di un progetto bello, mock-up              |
| `hero-3.jpg`      | Foto del team al lavoro, meeting, collaborazione      |
| `hero-4.jpg`      | Foto "behind the scenes", shooting fotografico        |
| `hero-5.jpg`      | Grafica o progetto completato (opzionale)             |
| `hero-6.jpg`      | Qualsiasi altra foto evocativa (opzionale)            |

**Formato consigliato:** JPG o PNG  
**Dimensioni consigliate:** almeno 800×600px  
**Orientamento:** meglio paesaggio (orizzontale) o quadrato  

---

## 🗂️ PORTFOLIO — Social Media (Giuliano)
📂 Cartella: `images/portfolio/social/`

Carica **1 immagine per ogni progetto** (fino a 5):

| Nome file            | Contenuto                                              |
|----------------------|--------------------------------------------------------|
| `social-1.jpg`       | Screenshot feed Instagram, post o risultati cliente 1  |
| `social-2.jpg`       | Screenshot feed Instagram, post o risultati cliente 2  |
| `social-3.jpg`       | Screenshot feed Instagram, post o risultati cliente 3  |
| `social-4.jpg`       | Screenshot feed Instagram, post o risultati cliente 4  |
| `social-5.jpg`       | Screenshot feed Instagram, post o risultati cliente 5  |

**Formato consigliato:** JPG o PNG  
**Dimensioni consigliate:** almeno 1200×800px (ratio 16:9)  
**Tip:** Puoi usare uno screenshot del profilo, una raccolta di post, o un mock-up su device  

> ⚠️ Dopo aver caricato le foto, dicci il nome/settore di ogni cliente
> così aggiorniamo titolo e descrizione nel sito.

## 🌐 PORTFOLIO — Siti Web (Nicholas)
📂 Cartella: `images/portfolio/web/`

Carica **1 immagine** per il sito realizzato:

| Nome file        | Contenuto                                              |
|------------------|--------------------------------------------------------|
| `web-1.jpg`      | Screenshot del sito realizzato (full page o hero)      |

**Formato consigliato:** JPG o PNG  
**Dimensioni consigliate:** almeno 1400×900px  
**Tip:** Puoi usare un mockup su schermo (es. su mockupworld.co gratis)

## 🏢 CLIENTI — Loghi Marquee
📂 Cartella: `images/clients/`

Carica un logo per ogni cliente con questi nomi:

| Nome file           | Note                                      |
|---------------------|-------------------------------------------|
| `client-01.png`     | Logo cliente 1 — sfondo trasparente       |
| `client-02.png`     | Logo cliente 2 — sfondo trasparente       |
| `client-03.png`     | Logo cliente 3 — sfondo trasparente       |
| `client-04.png`     | ...                                       |
| `client-XX.png`     | Continua con la numerazione               |

**Formato consigliato:** PNG con sfondo trasparente  
**Dimensioni consigliate:** circa 300×150px (proporzione logo)  
**Altezza consigliata:** max 80px di altezza effettiva del logo  

> ⚠️ Se non hai i loghi in PNG trasparente, puoi convertirli su remove.bg

## 🔠 LOGO AZIENDALE
📂 Posizione: `images/logo.png` oppure `images/logo.svg`

- **Formato consigliato:** SVG (scalabile) o PNG trasparente
- **Dimensioni:** almeno 400px di larghezza
- Verrà usato nella navbar e nel footer
